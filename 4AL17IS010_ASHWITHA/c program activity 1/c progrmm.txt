void distict_elements(int a[], int n)
{
    int i, j;
    // Pick all elements one by one
    for (i=0; i<n; i++)
    {
        // Check if the picked element is already printed
        for (j=0; j<i; j++)
        {
            if (a[i] == a[j])
                break;
        }
        // If not printed earlier, then print it
        if (i == j)
            printf(“%d “, a[i]);
    }
}

algorithm:

Algorithm to print distinct numbers in an array


1)Declare and input the array elements.

2)Traverse the array from the beginning.

3)Check if the current element is found in the array again.
If it is found, then do not print that element.
Else, print that element and continue.
