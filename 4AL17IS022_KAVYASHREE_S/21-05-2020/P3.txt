Harshad Number
Write a C++ program to find whether the given number is a Harshad number or not. Note that Harshard number is an integer that is divisible by the sum of its digits.
INPUT & OUTPUT FORMAT:

Input consist of 1 integer. If the given number is a Harshad Number, display "Harshad Number" or display "Not Harshad Number".

SAMPLE INPUT:

1729

SAMPLE OUTPUT:

Harshad Number

PROGRAM:

#include<iostream>
using namespace std;
int main()
{
  //Type your code here.
  int num;
  int temp;
  cin >> num;
  int sum = 0;
  temp = num;
  while(temp)
  {
    sum += temp % 10;
    temp = temp / 10;
  }
  int res = num % sum;
  if(res == 0)
    cout <<"Harshad Number";
  else
    cout <<"Not Harshad Number";
  return 0;
}
